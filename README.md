# Employee Management Dashboard

## Project Overview

The Employee Management Dashboard is a React-based web application that allows users to manage employee information efficiently.  
It provides features such as authentication, employee listing, adding and editing employee details, profile image upload with preview, filtering, searching, and printing employee data.

The application is built with a focus on clean UI, reusable components, and modern frontend development practices.

---

## Tech Stack Used

- **React (v19)** – Frontend library  
- **Vite** – Development and build tool  
- **Material UI (MUI)** – UI components and styling  
- **TanStack Table (React Table v8)** – Employee table implementation  
- **React Router DOM** – Routing and protected routes  
- **JavaScript (ES6+)** – Core language  


## Steps to Run the Project Locally

### Prerequisites

- Node.js (v18 or above)
- npm

### Installation & Run

```bash
npm install
npm run dev
```

## The application will be available at:

> http://localhost:5173
